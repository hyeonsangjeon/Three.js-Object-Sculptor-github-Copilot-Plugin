# Changelog

All notable changes to `threejs-sculpt-dna` are documented here.

## [Unreleased]

### Added

- Dependency-free deterministic visual regression matrix for a base sculpt and
  promoted curated variants, with fixed viewpoint plans, expected evidence
  paths, SHA-256 freshness, latest-per-pass review precedence, required layer
  scores, selected semantic feature gates, stable JSON classification, and
  explicit CLI exit codes.
- Additive manifest/report schema documentation that keeps AI vision as final
  authority and leaves existing `ObjectSculptSpec`, production, release, and
  evidence gates unchanged.

## [0.4.3] - 2026-07-12

### Added

- `sculpt-dna-variants` with deterministic semantic controls, invariants, provenance, and Coverage Curator.
- Evidence-backed production gates and explicit non-promotable preview mode.
- Repolis Living Archive flagship with generated branch geometry, runtime PBR, 2,600 instanced leaves, moss, code glyphs, energy paths, sockets, colliders, and destruction groups.
- Reproducible browser capture pipeline for hero PNG, rotating GIF, staged screenshots, comparison evidence, and SHA-256 artifact manifest.
- Original-to-variants-to-flagship README progression.
- Brick Off-Road and Seoul Palace challenge preview families.
- Copilot marketplace manifest and complete installation/user guide.
- GitHub Pages interactive demo.

### Changed

- Renamed the public repository to `threejs-sculpt-dna`.
- Branded the project as `threejs-sculpt-dna — A GitHub Copilot Plugin`.
- Strengthened SKILL reference linking for marketplace validation.
